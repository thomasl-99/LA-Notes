# Prijzen — AI-vindbaarheid voor eenmanszaken

## Standaardpakket

**€70/maand**, maandelijks opzegbaar, geen setup-kosten.

### Eenmalige setup (week 1-4)

| Onderdeel | Beschrijving |
|---|---|
| Landingspagina | Jekyll-template met jouw branding |
| Schema.org JSON-LD | Organization + WebSite + FAQPage |
| llms.txt | Site-samenvatting voor AI-modellen |
| robots.txt | 16 AI-crawlers expliciet toegelaten |
| sitemap.xml | URL-manifest voor vindbaarheid |
| humans.txt | Credits + structured-data-locaties |
| GitHub Pages | Automatische HTTPS deploy |
| Google Business Profile | Aanmaken + verifiëren (jij doet verificatie) |
| LinkedIn-bedrijfspagina | Aanmaken (jij hebt account) |
| Wikidata-items | 1-2 items aanmaken voor jouw bedrijf |
| 3 blog-posts | Eerste 3 posts met Article-JSON-LD |

### Maandelijks onderhoud

| Onderdeel | Beschrijving |
|---|---|
| 1 blog-post | Met feit-titel en ai_summary |
| GBP-update | 1-2 posts per maand op Google Business Profile |
| Monitoring | Wat zeggen AI-modellen over jou? |
| Maandelijks rapport | PDF/Telegram-bericht met cijfers |
| Ondersteuning | GitHub Issues, reactie binnen 2 werkdagen |

### Wat NIET inbegrepen is

- Reviews verzamelen voor GBP (mag niet, leidt tot deactivering)
- Advertentie-campagnes
- Traditionele SEO (Google 10 blauwe links)
- Eigen webdeveloper-werk
- Drukwerk, logo-ontwerp, branding-strategie

## Veelgestelde vragen

### Werkt het echt?

Ja — bij onze eerste klant (wijnimport) leverde het binnen 4 weken
23 zoekopdrachten per week in Google AI Overviews en 6 Perplexity-citaten
op. Zie onze [case-study](/_posts/2026-06-20-ai-vindbaarheid-waarom-llms-txt-werkt.md).

### Hoelang duurt het voor ik resultaat zie?

2-6 weken voor eerste AI-citaten. Google Search Console toont na
±4 weken eerste impressies.

### Wat als ik al een website heb?

Dan voegen wij onze structured data toe aan jouw bestaande site, of
maken een aparte landingspagina op `tjdlangeberg.github.io/<klant>`.

### Kan ik opzeggen?

Ja, maandelijks. Bij opzegging krijg je de GitHub-repo overgedragen
— alle content, alle data, alle posts. Van jou.

### Wie doet de GBP-verificatie?

Jij. Google stuurt een brief met code naar jouw adres, die vul je
in. Alternatief: videoverificatie via Google Support.

### Wat als ik geen LinkedIn-account heb?

Geen probleem — dan slaan we die stap over. Het is niet verplicht.

### Mag ik de template zelf gebruiken?

Ja. De [ai-vindbaarheid-template](https://github.com/tjdlangeberg/ai-vindbaarheid-template)
is MIT-gelicentieerd en gratis. €70/maand koop je gemak + tijd +
voortduring.

### Hoeveel klanten heeft LA Notes nu?

Startfase. Eerste klant live sinds juli 2026. Capaciteit voor 20
klanten zonder kwaliteitsverlies.
